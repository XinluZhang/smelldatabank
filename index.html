Synaesthesia Experiment (olfaction&vision)

This website is a collection of data on odor images and has now uploaded images of 50 different random odors, each of which will be presented using two types of images.

  • One type of image is data on the neurological response of the brain to different odors when inhaled by people under an EGG (Electroencephalogram) monitor. The image is mainly displayed through the α, β and θ values in the instrument. Visitors can select any of these odors to get an idea of their corresponding brainwave state.
  • Another is based on the data from the smell test, where we choose to record keywords with a high repetition rate in descriptions from different testers under the same smell and summarize them. The keywords are then fed into Midjourney’s program for AI painting, allowing each scent picture to be generated entirely through data programming.
